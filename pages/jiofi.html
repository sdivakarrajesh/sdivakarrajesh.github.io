My JioFi

What the App does?


Provides a dashboard for managing the JioFi device(JMR815)

Lets you perform admin level operations on the device such as filtering users, changing wifi SSID, password etc.

Key features


- Ability to block certain users with their mac Address
- Ability to block a connected user
- Change the Wifi SSID and password
- Know battery level
- Shows signal strength
- Details of connected users
- Connection info such as mobile number,operating band, frequency etc

Technical Aspects


- Making a HTTP request and async operations
- Session management with Cookies
- Responsive Animations and Transitions
- Storing and managing user preferences
- Transfering and managing data between Navigations and different screens

Why I made this App?


I used to have another hotspot device for which the vendor itself provided an application to manage the device using a mobile application.

However this JioFi device that I purchased recently does not have an app to manage it, and has only a web interface.

I couldn't find one. So I made one.

The App has received a positive feedback from the users in the Google Play store with more than 500 users in just 1 month and still growing!

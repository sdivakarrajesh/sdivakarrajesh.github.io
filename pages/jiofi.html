My JioFi

What the App does?


An Android app that allows what a traditional slambook allows a person to do

To share moments, favorites among collegues and friends at school

Key takeaways from the App


Does not require us to maintain a server. The app relies on the person's google account.

Uses Google's Drive and Sheets API to allow for the sharing of data between the users

Authentication is achieved via the Google+ API

What I learnt


How to make use of external API-s offered by other services and organization

How to read through documentation to make use of the API

How to integrate third party APIs to provide authentication of the users

Key moments in the App


The user signs into the App using Google sign-in.

Grants permission to the app to access Google Drive and Sheets

With a template of questions, the app uploads a spreadsheet to the user's Drive and provides a link (to record responses that is to be shared to the other recipients)

The user shares the link to the recipients

Using that link the recipient provides his responses for the questions and the app appends them to the sheet on the sender's Sheet
